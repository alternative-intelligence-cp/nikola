## üöÄ Quick Start

### Prerequisites

- **OS**: Ubuntu 24.04 LTS (required)
- **Compiler**: GCC 13+ or Clang 16+ (C++23)
- **Hardware**: AVX-512 CPU recommended, CUDA GPU optional
- **Memory**: 64GB RAM minimum (256GB recommended)

### Build Instructions

```bash
# Install dependencies
sudo apt update && sudo apt install -y \
    build-essential cmake libzmq3-dev libprotobuf-dev \
    liblmdb-dev libvirt-dev qemu-kvm libcurl4-openssl-dev

# Clone repository
git clone https://github.com/your-org/nikola.git
cd nikola

# Build
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release -DENABLE_AVX512=ON
make -j$(nproc)

# Run tests
ctest --output-on-failure

# Install
sudo make install
```

### Docker Deployment

```bash
# Set API keys
export TAVILY_API_KEY="your-key"
export FIRECRAWL_API_KEY="your-key"
export GEMINI_API_KEY="your-key"

# Start services
docker-compose up -d

# Check status
docker exec nikola-core /usr/local/bin/twi-ctl status
```

### CLI Usage

```bash
# Query the system
twi-ctl query "What is the golden ratio?"

# System status
twi-ctl status

# Trigger memory consolidation
twi-ctl nap

# Export to GGUF
twi-ctl export nikola-snapshot.gguf

# View metrics
twi-ctl metrics
```
