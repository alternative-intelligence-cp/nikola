## HOW TO FIND CONTENT
================================================================================

### ✅ RECOMMENDED APPROACH

1. **Use Keyword Search (Ctrl+F)**
   - Search for concepts, NOT section numbers
   - Examples:
     * "Wave Interference" instead of "Section 2.2"
     * "Mamba-9D" instead of "Section 3.2"
     * "ZeroMQ Spine" instead of "Section 4.1"
     * "Bootstrap" for IMP-03 system initialization
     * "Holographic Lexicon" for IMP-02 wave decoding
     * "Working Memory" for COG-04 refractive trapping
     * "PIMPL" for IMP-04 ABI stability

2. **Look for Section Markers**
   ```
   ================================================================================
   SECTION 2.2: Wave Interference Physics
   ================================================================================

   <!-- SOURCE: 02_foundations/02_wave_interference_physics.md -->
   ```

3. **Check Source Comments**
   - Every section includes `<!-- SOURCE: path/to/file.md -->`
   - Trace back to original specification file if needed

4. **Use Bug Sweep Markers**
   - Updated sections marked with [UPDATED] or [Bug Sweep XXX]
   - Examples: [Bug Sweep 012], [Bug Sweep 013], etc.

### ❌ AVOID THESE APPROACHES

- Don't rely on internal section numbers (like ## 6.1 or ## 4.2)
- Don't try to calculate which part file contains a section
- Don't search by section numbers from the table of contents
- Internal numbering ≠ TOC numbering (this causes confusion)
